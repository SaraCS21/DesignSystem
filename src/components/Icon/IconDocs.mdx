# Icon

`my-icon` es un componente de icon personalizable construido utilizando `LitElement`.

<Primary />

Este componente permite ajustar diversas propiedades como el color o el texto que contiene.

## Props

<Controls />

## Example Usage

```html
<my-icon
  name="arrow"
  color="blue"
  colorIntensity="500"
  arrowType="up"
  arrowSize="normal"
/>
```
